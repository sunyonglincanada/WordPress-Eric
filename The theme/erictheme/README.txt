== Description ==

Eric WordPress Theme
1. Libraries used: Bootstrap 4
2. Page Templates, Front Page, Blog Page, Post Detail Page(single.php)
3. Header, Sticky Footer

== Installation ==

1. In your admin panel, go to Appearance -> Themes and find Eric Theme.
2. Click on the 'Activate' button to use Eric Theme right away.

== Setup ==

1. In your admin dashboard, go to Pages -> Add New to create Welcome Page(static front page), About Us(select about us page template) and Blog pages.
2. Navigate to Posts -> Add New or insert some test posts.
3. Navigate to Settings > Reading in your admin dashboard and set your homepage displays option as a static page with Homepage(Welcome Page) and Posts page(Blog Page).
4. Enjoy!
