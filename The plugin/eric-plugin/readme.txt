== Description ==

Eric WordPress Plugin
1. This plugin adds a fixed sentence to the bottom of each post.
2. In your admin dashboard menu, Click Eric Plugin admin menu to see the content of Eric Plugin.

== Installation ==

1. In your admin dashboard, go to Plugins and find Eric Plugin.
2. Click on the 'Activate' button to use Eric Plugin right away.

== Uninstall ==

1. In your admin dashboard, go to Plugins and find Eric Plugin.
2. Click on the 'Deactivate' button to deactivate Eric Plugin right away.
